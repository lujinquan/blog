憧憬一段任性的旅行，最好在青春飞扬的时节，最好有个遥远美丽的目的地，最好能伴阳光前行，最好，有你。

一个人的旅行，反而会更贴近自己的内心，真正的停下来，享受自我的体验时刻，也许浮光掠影，走马观花之外，这才是深入体验，探索自我的最佳时间，纵然走过那么多城市，对于未知的风景，还是好奇。继续期待我的下一个旅行，拿起背包，感受不同地域不一样的节奏与表象。
